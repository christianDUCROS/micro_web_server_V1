Langage Micropython



MicroPython est une implémentation légère et efficace du langage de programmation Python 3 qui inclut un petit sous-ensemble de la bibliothèque standard Python et est optimisée pour fonctionner sur des microcontrôleurs et dans des environnements contraints.

MicroPython vise à être aussi compatible que possible avec Python normal pour vous permettre de transférer facilement du code du bureau vers un microcontrôleur ou un système embarqué.


Conclusion : Le micropython est le portage du langage Python sur micro-contrôleur.


Micropython est multi-matériel : Micropython est un langage interprété qui va pouvoir être installé sur de très nombreuses cartes.
