Site web embarqué sur microcontrôleur


Le serveur web fonctionne sous micropython

Connecter à un smartphone en mode Acces Point par exemple

Il permet d'afficher des pages web (html) , gère les fichiers css, les liens internes et externes , les images png, jpeg, jpe, svg,webp...

Attention la taille d’une page est très limitée (17ko environ)

Par défaut les fichiers devront être stockés sous www

Par défaut c'est la page index.html qui est ouverte
